## 开发环境

### 启动开发服务器

```bash
npm run dev
```

服务器将在固定端口启动：**http://localhost:8080**

如果端口 8080 被占用，需要先释放该端口或修改 `vite.config.ts` 中的配置。

### 其他命令

- `npm run build` - 构建生产版本
- `npm run preview` - 预览生产构建
